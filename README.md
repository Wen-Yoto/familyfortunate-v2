## Run Locally

1. Download or clone the project

   ```
   git clone https://github.com/joaquinvaldezzz/familyfortunate.git
   ```

2. Go to the project directory

   ```
   cd familyfortunate
   ```

3. Install dependencies

   ```
   yarn install
   ```

4. Start the local server

   ```
   yarn dev
   ```
