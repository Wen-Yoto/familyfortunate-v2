## Run Locally

1. Download or clone the project

   ```
   git clone https://github.com/atweberlly/familyfortunate-v2.git
   ```

2. Go to the project directory

   ```
   cd familyfortunate-v2
   ```

3. Install dependencies

   ```
   npm install
   ```

4. Start the local server

   ```
   npm run dev
   ```
